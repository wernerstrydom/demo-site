Featured Projects

Here are some of the personal projects I've worked on over the years. They range from web applications to system utilities, hardware experiments, and explorations of computing history.

This Website

Technologies: HTML 4.01, CSS, JavaScript

Year: 2026

This website itself is a project in progressive enhancement. Built to work on browsers from 1999 onwards while providing a modern experience on current browsers, it demonstrates how web standards and thoughtful design can create sites that are both accessible and future-proof.

The site uses no build tools, frameworks, or dependencies. It's pure HTML, CSS, and minimal JavaScript, structured in layers so that each browser gets the best experience it can handle. The source code is intentionally readable and educational.

Key Features:

  • Works on Netscape Navigator 4, Internet Explorer 5, and all modern browsers
  • Progressive CSS with fallbacks for older browsers
  • Responsive design using media queries (gracefully ignored by old browsers)
  • Semantic HTML that works without CSS or JavaScript
  • Fast loading with minimal resources

Cross-Platform Build System

Technologies: Shell scripts, Make, Docker

Year: 2025

A build system for compiling software across multiple architectures including x86-64, ARM, PowerPC, and MIPS. This project grew out of the need to build modern software for vintage systems in my home lab.

The system uses Docker containers with cross-compilation toolchains, automated testing on real hardware, and a simple interface for building packages. It's helped me port several open-source applications to architectures that are no longer officially supported.

Supported Architectures:

  • x86-64 (Linux, BSD)
  • ARM (32-bit and 64-bit)
  • PowerPC (32-bit and 64-bit)
  • MIPS (big-endian and little-endian)
  • SPARC (32-bit and 64-bit)

Vintage Browser Testing Suite

Technologies: Python, Selenium, VirtualBox

Year: 2024

An automated testing suite for verifying web compatibility across vintage browsers. The system uses virtual machines running period-appropriate operating systems and browsers to test websites for compatibility.

This tool has been invaluable for ensuring that websites work on older browsers without requiring manual testing on physical hardware. It includes screenshot comparison, HTML validation, and accessibility checks.

Tested Browsers:

  • Netscape Navigator 4.x (1997-1998)
  • Internet Explorer 5.x (1999-2001)
  • Mozilla 1.0 (2002)
  • Opera 5-7 (2000-2003)
  • Safari 1.0 (2003)

Home Lab Monitoring Dashboard

Technologies: Python, Flask, JavaScript, SQLite

Year: 2024

A lightweight monitoring dashboard for tracking the health and performance of home lab systems. It collects metrics from diverse systems including modern Linux servers, vintage Unix workstations, and network equipment.

The dashboard is designed to work with minimal resources and supports systems that can't run modern monitoring agents. It uses standard protocols like SNMP, SSH, and HTTP to gather data, and presents it in a simple, accessible interface.

Monitored Metrics:

  • CPU, memory, and disk usage
  • Network traffic and errors
  • System uptime and load
  • Temperature sensors (where available)
  • Service availability and response times

Static Site Generator for Progressive Enhancement

Technologies: Python, Jinja2, Markdown

Year: 2023

A static site generator specifically designed for creating progressively enhanced websites. Unlike most static site generators that target modern browsers, this tool generates HTML 4.01 markup with progressive CSS and JavaScript layers.

The generator includes templates for common page types, automatic generation of accessible navigation, and validation against HTML 4.01 standards. It's been used to create several personal and client websites that need maximum compatibility.

Features:

  • Markdown to HTML 4.01 conversion
  • Automatic progressive CSS generation
  • Accessibility checking and validation
  • Support for multiple output formats
  • No JavaScript required for the build process

Retro Computing Blog Archive

Technologies: HTML, CSS, Archive.org API

Year: 2023

A curated archive of articles, documentation, and resources about vintage computing systems. The archive focuses on preserving knowledge about systems that are no longer in mainstream use but remain historically and technically interesting.

Content is organized by architecture, manufacturer, and topic, with an emphasis on practical information for people still using or maintaining these systems. The site itself is built with progressive enhancement to ensure it's accessible on the very systems it documents.

Covered Topics:

  • PowerPC systems (Apple, IBM, Amiga)
  • MIPS workstations (SGI, DEC)
  • SPARC servers (Sun Microsystems)
  • Classic Mac OS and Mac OS X
  • Vintage Unix systems

Experimental Projects

HTML Email Templates

Accessible Form Components
