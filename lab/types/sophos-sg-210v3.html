Overview

The Sophos SG 210 rev.3 is a 1U rackmount x86 hardware platform originally sold as a unified threat management (UTM) appliance. Sophos manufactured it in 2014-2016. The hardware uses standard x86 components. It runs any x86-64 operating system.

Sophos discontinued the bundled UTM software in 2021. The hardware remains functional. This makes it suitable for learning platform deployment, testing alternative firewall software, and understanding the gap between vendor appliances and open-source solutions.

Hardware Specifications

Component Specification
CPU Intel Atom C2358 dual-core @ 1.7GHz
Memory 4GB DDR3 (soldered, non-upgradeable)
Storage 64GB mSATA SSD
Network Ports 8x GbE (Intel i350 quad-port cards)
Expansion 1x PCIe slot (low-profile)
Power 60W, 12V DC external PSU
Dimensions 1U rackmount, 280mm depth
BIOS AMI BIOS, standard x86 boot

Capabilities

The Intel Atom C2358 handles routing and firewall workloads. Throughput depends on the software and enabled features. The CPU supports AES-NI for hardware-accelerated encryption.

The 8 Intel i350 NICs provide flexibility for network topology. Each port supports VLANs. The i350 has mature Linux driver support. The hardware handles multiple VLANs without performance degradation.

The 4GB RAM limits concurrent connection tracking. For typical home lab or small office use, this is sufficient. High-traffic scenarios require tuning connection tracking parameters or accepting limitations.

Operating System Compatibility

The hardware runs standard x86-64 operating systems. Tested and working:

  • Ubuntu Server 22.04, 24.04
  • Debian 11, 12
  • pfSense 2.7
  • OPNsense 24.1
  • VyOS 1.4

Installation requires USB boot. The BIOS supports legacy boot and UEFI. The mSATA SSD appears as a standard SATA device to the installer.

Network interface naming varies by OS. Ubuntu uses predictable names (enp2s0, enp3s0, etc.). pfSense uses em0-em7. This matters for configuration scripts and documentation.

Known Hardware Issues

The Intel Atom C2000 series has a clock signal degradation defect. Affected CPUs fail after 18-24 months of continuous operation. Intel documented this as erratum AVR54. Sophos issued hardware replacements during the warranty period.

Units manufactured after mid-2015 include the hardware fix. Units manufactured before the fix may fail without warning. No software mitigation exists. The failure is permanent.

The external power supply uses a proprietary connector. Replacements are difficult to source. The device requires 12V DC at 5A. Standard 12V supplies work with the correct barrel connector (5.5mm outer, 2.5mm inner, center positive).

Original Software (Sophos UTM 9)

Sophos shipped the hardware with UTM 9 firmware. The software provided firewall, VPN, IPS, web filtering, and application control through a web interface.

Sophos declared UTM 9 end-of-life in 2021. No security updates. No feature updates. No technical support. The software is obsolete.

The hardware outlived the software. This is common with vendor appliances. The vendor controls the software lifecycle. The hardware lifecycle is independent but constrained by vendor support.

Use Cases

The hardware serves as a platform for learning firewall deployment. Installing Ubuntu or pfSense exposes the underlying mechanisms that commercial appliances abstract. This includes network interface configuration, routing tables, firewall rules, and NAT.

The 8-port configuration supports complex network topologies. VLAN segmentation, DMZ networks, and multi-WAN setups are possible. The hardware handles scenarios that single-NIC systems cannot.

The platform demonstrates that vendor appliances are x86 hardware running Linux. The value proposition is the software, support, and integration. The hardware is commodity.

Limitations

The Atom CPU limits throughput for compute-intensive tasks. IPS, deep packet inspection, and VPN encryption reduce throughput significantly. For basic routing and stateful firewall, the CPU is adequate.

The 4GB RAM is soldered. No upgrade path. Connection tracking tables consume RAM. High connection counts require tuning or accepting connection limits.

The lack of IPMI or out-of-band management complicates remote administration. Serial console access requires physical presence or a console server.

Acquisition

These units appear on the used market. Prices range from $50-150 depending on condition and included accessories. Verify the manufacturing date to avoid units with the C2000 clock defect.

Check for included power supply. Replacement supplies are expensive. Verify all 8 network ports function. Failed NICs reduce the platform's value.

Documentation

Sophos archived the original UTM 9 documentation. The hardware guide covers physical specifications. This information remains relevant for understanding the platform.
