# Personal Website with Progressive Enhancement

A personal website built using progressive enhancement principles to work on browsers from 1999 to the present day and beyond.

## Overview

This website demonstrates how progressive enhancement can create sites that work on virtually any browser while still providing a modern, polished experience on current browsers. It works on Netscape Navigator 4, Internet Explorer 5, and all modern browsers.

## Features

- **Universal Compatibility**: Works on browsers from 1999 onwards
- **No Build Tools**: Pure HTML, CSS, and JavaScript - no compilation needed
- **No Dependencies**: No frameworks or external libraries
- **Responsive Design**: Adapts to different screen sizes on modern browsers
- **Dark Mode Support**: Respects user's color scheme preference
- **Accessible**: Semantic HTML and keyboard navigation
- **Fast Loading**: Less than 50KB total page size
- **Print-Friendly**: Optimized for printing

## Pages

- **index.html** - Home page with introduction
- **opinions.html** - Thoughts on technology and web standards
- **homelab.html** - Home lab hardware and experiments
- **projects.html** - Personal projects showcase
- **about.html** - About page with technical details

## Technical Details

- HTML 4.01 Transitional for maximum compatibility
- CSS with progressive enhancement using @supports
- Minimal, unobtrusive JavaScript with feature detection
- Works without CSS or JavaScript
- Tested on vintage browsers (Netscape 4, IE5) and modern browsers

## Deployment

Simply serve these files from any web server. No server-side processing required.

## Philosophy

Built with progressive enhancement principles inspired by "Adaptive Web Design" by Aaron Gustafson. The site works everywhere and enhances based on browser capabilities.
