What You'll Find

Writing contains technical opinions and analysis. Topics include automation, configuration management, security practices, and the gap between theory and implementation. The goal is helping readers trace connections between systems.

Technical Notes

This site uses HTML 4.01 Transitional for maximum compatibility. CSS applies progressive enhancement through @supports queries. JavaScript is minimal and optional. The site works without CSS or JavaScript.

Modern browsers get responsive layouts, custom properties, and smooth transitions. Older browsers get clean, readable text with basic formatting. Both experiences are intentional.
