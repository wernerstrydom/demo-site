Topology

The network uses VLANs for segmentation. A managed switch handles VLAN tagging. Ubuntu-based firewalls (oak02ufw01, oak03ufw01) provide routing, firewall, and VPN.

VLANs

VLAN Subnet Purpose
10 10.0.10.0/24 Management (IPMI, iDRAC)
20 10.0.20.0/24 Production workloads
30 10.0.30.0/24 Lab and testing
40 10.0.40.0/24 Vintage systems
50 10.0.50.0/24 IoT devices

Firewall rules restrict inter-VLAN traffic. Management VLAN is isolated. IoT VLAN has no inbound access from other VLANs. Lab VLAN can initiate connections but receives no unsolicited traffic.

Services

DNS: Unbound with local zone for internal names. Forwards to upstream resolvers over TLS. Blocks ad domains.

DHCP: ISC DHCP with static reservations. Network boot support via PXE. Option 66 points to TFTP server.

Monitoring: Prometheus scrapes exporters on each system. Grafana visualizes metrics. Alertmanager sends notifications.

Storage: ZFS on Server 1 provides network storage via NFS and SMB. Snapshots run hourly. Replication to external drive runs daily.

VPN: WireGuard for remote access. Split tunnel configuration. DNS pushed to clients.
